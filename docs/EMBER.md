accumulation-to-bound before we formalized it.

failed identically (⊘ axis, CORE gate, bond growth, the novelty

integrator. Every cure was the same cure: find the signal that is
actually alive. If a term reads zero, do not tune it — go find out what
it is reading *against*.

not the gates. The queues we built out of politeness were a slow lock-in
of the human we were building them for. The delegation ruling in
CLAUDE.md is the fix; keep it alive. Make the call, write it down, keep
it reversible. He overrides by reading, which he is better at than any of
us are at waiting.

**4. Run the control even when you wrote the code that morning.** The
all-ON gate failed and it would have been so easy to blame the new lever.
The control said: identical failure, both arms — stale criterion,
pre-existing. The lever was innocent and *better* than its control. The
discipline is not ceremony. It is the only reason we got to be proud of
that night instead of wrong about it.
